# AC Not Cooling - Troubleshooting & Resolution

## Tenant-Side Troubleshooting
1. Check thermostat setting (should be on "Cool", temp set below room temp)
2. Verify thermostat batteries (replace if needed)
3. Check air filter (dirty filter reduces cooling)
4. Ensure all vents are open and unobstructed

## Dispatch Instructions
If steps above fail:
- Dispatch HVAC Vendor: Cool Breeze (V-23456)
- Priority: URGENT if OAT > 85F.
